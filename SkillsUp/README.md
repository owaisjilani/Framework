# SkillsUp

Interview preparation and past interview practice programs.

## Structure

- `src/com/programs/` - Java interview coding programs and company-specific practice files.
- `src/Utils/` - reusable Java utility examples.
- `interview-notes/` - text notes, SQL snippets, Selenium notes, and problem statements.
- `documents/` - interview-related PDF documents.

Generated files and IDE workspace metadata are intentionally ignored so the repository stays focused on source code and notes.
